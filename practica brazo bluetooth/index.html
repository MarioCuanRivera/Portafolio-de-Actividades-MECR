Brazo Bluethoot

Mario E. Cuan Rivera, Sebastián Rodríguez Zapata, Juan Pablo Sotres Camacho
Ing. Mecatrónica
Fecha: Noviembre 02, 2024
INT MECATRÓNICA
Arduino
Progrmacion
Monitor Bluethoot

- Resumen -

En esta práctica, se diseñó y construyó un brazo robótico controlado por Bluetooth. Este brazo es capaz de moverse en diferentes direcciones y realizar acciones como agarrar y soltar objetos.

Se utilizó Arduino para programar los movimientos y una aplicación móvil para el control remoto.

Este proyecto fomenta el aprendizaje práctico de control remoto y robótica aplicada.

- Desarrollo -

El desarrollo de brazos robóticos ha sido crucial en la automatización industrial y educativa. Este proyecto busca introducir conceptos básicos de mecatrónica, control inalámbrico y programación.

El objetivo principal es construir un brazo funcional que pueda ser operado a distancia mediante una interfaz móvil, empleando tecnologías accesibles y de bajo costo.

- Materiales -

-Arduino Uno

-Módulo Bluetooth HC-05

-Servomotores

-Fuente de alimentación

-Brazo previamente hecho

- Diseño -

Se creó un modelo 3D preliminar del brazo robótico para definir sus dimensiones y funcionalidad.

El diseño incluye una base giratoria, una articulación en el codo y una pinza para agarrar objetos.

La comunicación se planificó mediante Bluetooth, con comandos enviados desde un dispositivo móvil.

Usamos el Diseño del brazo que hicimos el primer parcial porque nada mas modificamos para poder controlarlo con bluetooth.

- Programacion-

Se programaron los movimientos del brazo con comandos específicos para cada articulación con arduino uno y con la aplicacion "MIT App Inventor"

Video

- Resultados-

El brazo robótico se controló exitosamente desde un teléfono móvil, realizando movimientos precisos en sus articulaciones.

- Conclusiones -

La práctica logró los objetivos establecidos al combinar conceptos de robótica, programación y control remoto.

Este proyecto demostró la viabilidad de integrar tecnologías accesibles para construir sistemas automatizados. Como mejora, se podría optimizar la estructura para aumentar su capacidad de carga o implementar sensores para mayor precisión.
